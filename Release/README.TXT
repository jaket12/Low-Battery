Thanks for trying our game out. It was built from the ground up in a weekend during the Ludum Dare Game Jam 39. Let us know what you think!

You can run the game by opening the file "LowBattery.jar".

Controls:
Movement: Arrow keys
Interact: Enter or A button
Exit Game: Escape
